## 📋 About This Project

This is my professional portfolio website showcasing my work as a **Virtual Systems Architect**, **Virtual Assistant**, and **Full-Stack Developer**. The site is built with a focus on the philosophy **"Architecture over Hustle"** — emphasizing structured, scalable digital frameworks over chaotic, reactive work.

### ✨ Key Features

- 🎨 **Modern Dark Theme** — Charcoal background with teal/cyan accent colors
- ✨ **Interactive Animations** — Typewriter effect, particle background, bubble rise effect
- 🖱️ **Custom Cursor & Glow Trail** — Enhanced user experience with mouse interactions
- 📱 **Fully Responsive** — Optimized for all devices (mobile, tablet, desktop)
- 🖼️ **Auto-Sliding Gallery** — Showcase of work with detail modals
- 📝 **Testimonial Carousel** — 13 client and partner testimonials
- 🏷️ **Service Case Studies** — Detailed breakdowns of each service pillar
- 🔗 **Social Media Integration** — Links to LinkedIn, GitHub, Instagram, X (Twitter), WhatsApp
- 📧 **Contact Form** — Integrated with Formspree for message handling
- 📄 **Resume Download** — Direct PDF download from the site

📝 Customization Guide
Update Personal Information
Profile Photo: Replace assets/joshua.png

OG Image: Update assets/og-image.png (1200x630px)

Resume: Replace assets/Joshua_Resume.pdf

Contact Email: Update in index.html and script.js

Phone Number: Update in index.html

Social Links: Update in contact section

Update Social Preview Image
Create a 1200x630px image

Save as og-image.png in assets/

Update the URL in index.html if needed

Update Gallery Images
Add new images to assets/ folder

Update gallery data in script.js

Add new gallery cards in index.html

🐛 Known Issues & Fixes
Issue	Solution
Social preview not showing	Check absolute URL in og:image tag
Gallery images not loading	Verify image names match exactly (case-sensitive)
Form not submitting	Check Formspree endpoint is correct
Mobile menu not working	Ensure nav-toggle ID is correct
